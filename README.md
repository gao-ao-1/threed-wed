# 新网站（示例静态站点）

这是一个非常简单的静态站点示例，包含 `index.html`、`style.css` 和 `script.js`。

快速预览：

- 在文件管理器中双击 `index.html` 打开。
- 或者在项目根目录运行简单静态服务器：

```bash
# 全局安装 http-server（如未安装）
npm install -g http-server
# 或使用 npx 临时运行
npx http-server .
```

然后在浏览器打开 `http://localhost:8080`（端口可能不同）。

可扩展：添加更多页面、图片或框架（如 Vite / Vue）。
